Run the full close-day workflow: review today, roll tasks, and capture a session log.

## Steps

1. Open today's daily note (`00_Daily/YYYY-MM-DD.md`).
2. **Evening Review** — prompt for and fill in:
   - Wins (what went well)
   - Energy level (1–5)
   - What drained you
   - One thing to carry forward tomorrow
3. **Task Rollover** — find all `- [ ]` items in today's note. Copy them to tomorrow's note under `## Carried Forward`. Add source link: `(from [[YYYY-MM-DD]])`.
4. **Idea Triage** — list any raw ideas captured today (lines with `#idea`). For each, ask: *inbox capture or discard?* Create inbox notes for keepers.
5. **Session Log** — prompt to run `/compress` (or auto-run it if I've confirmed that preference). If skipped, add a `#no-log` tag to today's note as a reminder.
6. **Tomorrow Primer** — create tomorrow's daily note stub with carried tasks and an empty `## Morning Focus`.
7. Sign off: *"Day closed. {{N}} tasks carried, {{N}} ideas captured."*
